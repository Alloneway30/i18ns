# Pedido de Chave

A Extensão Immersive Translate suporta muitos serviços de tradução, alguns deles exigem que você solicite a chave API do serviço correspondente antes de poder usá-los. Compilei os seguintes tutoriais de aplicação para os serviços a partir da web. Se houver alguma omissão ou não forem atualizados a tempo, sinta-se à vontade para clicar no canto superior direito para editar estas páginas.
