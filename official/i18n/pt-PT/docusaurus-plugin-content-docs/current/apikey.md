# Solicitação de Chave

A Extensão de Tradução Imersiva suporta vários serviços de tradução, alguns deles exigem que você solicite a chave API do serviço correspondente antes de poder usá-los. Compilei os seguintes tutoriais de aplicação para os serviços da web, se houver alguma omissão ou atualização não realizada a tempo, por favor, sinta-se à vontade para clicar no canto superior direito para editar estas páginas.
