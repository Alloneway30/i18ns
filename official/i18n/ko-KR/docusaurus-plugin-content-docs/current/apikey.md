# 키 신청

몰입형 번역 확장 기능은 많은 번역 서비스를 지원하며, 이 중 일부 서비스는 해당 서비스의 API 키를 신청해야 사용할 수 있습니다. 저는 인터넷에서 다음 서비스의 신청 방법을 정리했습니다. 누락되었거나 시기적절하게 업데이트되지 않은 부분이 있다면, 오른쪽 상단의 페이지를 편집하여 알려주시기 바랍니다.
