# Заявка на ключ

Расширение для погружающего перевода поддерживает множество сервисов перевода, для использования некоторых из них необходимо подать заявку на API ключ соответствующего сервиса. Я собрал инструкции по подаче заявок на следующие сервисы из интернета. Если есть пропуски или информация не обновлялась вовремя, пожалуйста, нажмите на иконку редактирования в правом верхнем углу этих страниц.
