# Demande de Clé

L'extension Immersive Translate prend en charge de nombreux services de traduction, certains d'entre eux nécessitent que vous demandiez la clé API du service correspondant avant de pouvoir les utiliser. J'ai compilé les tutoriels de demande suivants pour les services depuis le web, si des omissions ou des mises à jour ne sont pas effectuées à temps, n'hésitez pas à cliquer dans le coin supérieur droit pour éditer ces pages.
