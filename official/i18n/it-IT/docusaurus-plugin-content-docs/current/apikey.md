# Richiesta di Chiave

L'estensione Immersive Translate supporta numerosi servizi di traduzione, alcuni dei quali richiedono di applicare per la chiave API del servizio corrispondente prima di poterli utilizzare. Ho compilato i seguenti tutorial di applicazione per i servizi dal web, se ci sono delle omissioni o non sono aggiornati in tempo, per favore sentiti libero di cliccare sull'angolo in alto a destra per modificare queste pagine.
