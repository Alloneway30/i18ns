# APIキーの申請

没入型翻訳エクステンションは、多くの翻訳サービスをサポートしていますが、一部の翻訳サービスでは、対応するサービスのAPIキーを申請して使用する必要があります。インターネットから以下のサービスの申請ガイドをまとめました。もし漏れがあったり、最新の情報に更新されていない場合は、右上の角をクリックして、これらのページを編集してください。
