# Solicitud de Clave

La Extensión de Immersive Translate soporta muchos servicios de traducción, algunos de ellos requieren que solicites la clave API del servicio correspondiente antes de que puedas usarlos. He compilado los siguientes tutoriales de aplicación para los servicios de la web, si hay alguna omisión o no se actualizan a tiempo, por favor, siéntete libre de hacer clic en la esquina superior derecha para editar estas páginas.
