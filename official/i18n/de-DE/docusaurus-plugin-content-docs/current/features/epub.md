# Zweisprachige Übersetzung für E-Books

Klicken Sie auf das Plugin-Symbol -> Mehr, und wählen Sie dann "E-Book lesen" (unterstützt gängige Formate wie Epub, Mobi) oder "Zweisprachiges Epub E-Book erstellen".
